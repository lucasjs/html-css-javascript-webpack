- Clone the repo

```bash
$ git clone https://github.com/robertoachar/html-css-javascript-webpack.git
```

- Install dependencies

```bash
$ npm install
```

- Run scripts

| Action                 | Usage           |
| ---------------------- | --------------- |
| Start development mode | `npm start`     |
| Lint code              | `npm run lint`  |
| Build for production   | `npm run build` |
